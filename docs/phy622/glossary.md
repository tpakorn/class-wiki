# Glossary

| Term | Definition |
|------|-----------|
| Action | Time integral of the Lagrangian; stationary for the true trajectory. |
| Analytic Function | Function differentiable in a neighborhood of every point in its domain. |
| Cauchy-Riemann | First-order PDEs linking real and imaginary parts of an analytic function. |
| Christoffel Symbols | Connection coefficients describing how basis vectors change. |
| Conformal Map | Angle-preserving transformation. |
| Contour | Piecewise-smooth path in the complex plane. |
| Differential Form | Antisymmetric tensor; integrates naturally over manifolds. |
| Euler-Lagrange | ODE whose solutions extremize a functional. |
| Functional | Map from functions to numbers. |
| Geodesic | Locally shortest path on a curved manifold. |
| Group | Set with associative binary operation, identity, and inverses. |
| Hamilton's Principle | Physical paths make the action stationary. |
| Lagrange Multiplier | Undetermined parameter enforcing a constraint. |
| Lie Algebra | Tangent space at identity of a Lie group with bracket. |
| Residue | Coefficient of $1/(z-z_0)$ in a Laurent expansion. |
| Simply Connected | Domain where every closed loop contracts to a point. |
| Stationary | First variation vanishes. |
