# Standing Waves

Standing waves form from superposed identical waves traveling in opposite directions.

## Intuition
A guitar string fixed at both ends vibrates in fixed patterns — standing waves.

## Formal Definition
Node spacing:
$$
\Delta x = \frac{\lambda}{2}
$$

Resonant frequencies on a string:
$$
f_n = \frac{n v}{2L}, \quad n = 1,2,3,\ldots
$$

## Worked Example
A $0.6\,\text{m}$ guitar string with wave speed $400\,\text{m/s}$ has fundamental $f_1 \approx 333\,\text{Hz}$.

## Common Mistakes
Forgetting that boundaries enforce nodes at fixed ends.
