# Rotation

Rotation describes motion about an axis using angular quantities analogous to linear ones.

## Intuition
A wheel rolls: each point has a different linear speed but the same angular speed.

## Formal Definition
Angular displacement $\theta$, velocity $\omega = d\theta/dt$, acceleration $\alpha = d\omega/dt$.

Rolling without slipping:
$$
v_{CM} = R\omega
$$

Kinetic energy of a rigid body:
$$
K = \tfrac12 I_{CM} \omega^2 + \tfrac12 M v_{CM}^2
$$

## Worked Example
A solid disk ($I = \tfrac12 MR^2$) rolling down an incline speeds up under both translation and rotation.

## Common Mistakes
Using $I = MR^2$ for every shape.
