# Vectors

Vectors encode both magnitude and direction, which many physical quantities require.

## Intuition
Displacements, velocities, forces, and fields cannot be described by a single number — direction matters.

## Formal Definition
A vector $\vec{A}$ has components $A_x$ and $A_y$ in 2D, or $A_x$, $A_y$, $A_z$ in 3D.

## Mathematical Formulation
$$
\vec{A} = A_x \hat{i} + A_y \hat{j} + A_z \hat{k}
$$

The magnitude:
$$
|\vec{A}| = \sqrt{A_x^2 + A_y^2 + A_z^2}
$$

## Worked Example
For $\vec{A} = 3\hat{i} + 4\hat{j}$, $|\vec{A}| = 5$.

## Common Mistakes
Forgetting direction when adding vectors algebraically.

Q: Is speed a vector?
