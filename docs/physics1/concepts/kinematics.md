# Kinematics

Kinematics describes motion without considering its causes.

## Intuition
Before asking *why* an object moves, describe *how* it moves.

## Formal Definition
Kinematics relates position, velocity, and acceleration using differentiation and integration.

## Mathematical Formulation
$$
\vec{v} = \frac{d\vec{x}}{dt}, \qquad \vec{a} = \frac{d\vec{v}}{dt}
$$

Constant acceleration equations:
$$
v = v_0 + at, \quad x = x_0 + v_0 t + \tfrac12 a t^2
$$

## Worked Example
A car accelerates at $2\,\text{m/s}^2$ from rest for $5\,\text{s}$. Final speed $v = 10\,\text{m/s}$.

## Common Mistakes
Using $v^2 = v_0^2 + 2ax$ with inconsistent signs.
