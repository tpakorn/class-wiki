# Glossary

| Term | Definition |
|------|-----------|
| Basis | A linearly independent spanning set for a vector space. |
| Bessel Function | Solution to a specific second-order ODE appearing in cylindrical problems. |
| Characteristic Polynomial | Polynomial whose roots are eigenvalues of a matrix. |
| Curl | Measure of local rotation of a vector field. |
| Delta Function | Generalized function zero everywhere except the origin. |
| Divergence | Net outflow per unit volume at a point. |
| Eigenvalue | Scalar factor by which an eigenvector is scaled under a linear map. |
| Eigenvector | Direction unchanged (except for scale) by a linear transformation. |
| Fourier Transform | Integral transform representing a function as a superposition of complex exponentials. |
| Gradient | Vector of partial derivatives; points in direction of greatest increase. |
| Green's Function | Solution to the delta-source problem used to construct general solutions. |
| Hermitian Matrix | Complex square matrix equal to its own conjugate transpose. |
| Jacobian | Determinant of the derivative matrix for a coordinate transformation. |
| Laplace Transform | Integral transform converting derivatives into multiplication by $s$. |
| ODE | Equation containing derivatives of a single independent variable. |
| PDE | Equation containing partial derivatives of multiple independent variables. |
| Vector Triple Product | Identities involving three vectors: $\vec{A}\times(\vec{B}\times\vec{C})$.  |
