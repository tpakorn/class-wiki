# Faraday's Law

*Source lecture(s):* [SC134 Lec11-13](../lectures.md)

## Physical Meaning

Changing magnetic flux induces EMF.

## Variable Definitions

| Symbol | Meaning |
|--------|---------|
| $\mathcal{E}$ | induced EMF |
| $\Phi_B$ | magnetic flux |
| $N$ | number of turns |

## Assumptions

Quasi-static (neglect radiation); $N$ turns linked by same flux.

## Derivation

Empirical law; minus sign from Lenz's law.

## Applications

Generators, transformers, induction stoves, guitar pickups.
