# Gauss's Law

*Source lecture(s):* [SC134 Lec4](../lectures.md)

## Physical Meaning

Net outward flux equals enclosed charge over $\varepsilon_0$.

## Variable Definitions

| Symbol | Meaning |
|--------|---------|
| $\Phi_E$ | electric flux |
| $q_{\text{enc}}$ | enclosed charge |
| $\varepsilon_0$ | permittivity |

## Assumptions

Valid for any closed surface; useful when symmetry is high.

## Derivation

Apply divergence theorem to Coulomb field; integrate over sphere.

## Applications

Capacitor design, shielding, coaxial cable fields.
