# Simulations & Interactive Learning

Interactive widgets (live) and proposals for future ones.

## Live

- **[Potential-flow builder](potential-flow-builder.md)** — superpose uniform flow,
  sources, vortices and doublets; watch streamlines assemble a cylinder or a lifting
  body in real time.

## Proposed

- **[Kelvin–Helmholtz billow tank](kh-billow-tank.md)** — animated interface roll-up
  with adjustable shear and stratification
- **[Energy-cascade sandbox](energy-cascade-sandbox.md)** — interactive $E(k)$ spectrum
  with $Re$ slider showing the inertial range grow

Each proposal specifies learning goals, inputs/outputs, and implementation notes so it
can be built as a standalone HTML/JS asset (like the live ones in
[`assets/`](https://github.com/tpakorn/class-wiki/tree/main/docs/fluids/assets)).
